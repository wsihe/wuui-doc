### 特性

- 提炼自中后台产品的交互语言和视觉风格。
- 开箱即用的 Vue2.0 组件。
- 基于 npm + webpack + babel 的工作流

### 支持环境

* 现代浏览器和 IE9 及以上
* 支持服务端渲染

### 已完成组件

- Grid 栅格
- icon 图标
- Button 按钮
- Input 输入框
- Radio 单选框
- Checkbox 多选框
- Select 选择
- AutoComplete 自动完成
- Switch 开关
- Pagination 分页
- Message 消息
- Dropdown 下拉菜单
- Tabs 标签页
- Collapse 折叠面板
