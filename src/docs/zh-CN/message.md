## Message 消息提示

## 何时使用

- 可提供成功、警告和错误等反馈信息。
- 顶部居中显示并自动消失，是一种不打断用户操作的轻量级提示方式。

## API

### this.$message

| 参数       | 说明           | 类型                       | 默认值       |
|------------|----------------|--------------------------|--------------|
| content    | 提示内容       | string\|ReactNode | -           |
| duration   | 自动关闭的延时，单位秒 | number               | 3          |
| onClose   | 关闭时触发的回调函数 | Function          | -         |
